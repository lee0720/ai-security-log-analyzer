🔐 AIによる不正アクセス検知システム
📌 概要

本プロジェクトは、サーバーログを解析し、不正アクセスや異常な通信を検知するWebアプリケーションです。
Streamlitを用いて可視化し、IsolationForestによる異常検知を行います。

🎯 背景・課題

サーバーログは大量に生成されるため、以下の課題があります：

手動での分析が困難
異常アクセスの見逃しリスク
攻撃の早期発見が難しい

👉 本ツールはこれらを解決するために開発しました。

💡 主な機能
📊 ログ可視化
IPアドレスごとのアクセス数表示
ステータスコード分布のグラフ化
🚨 エラーログ抽出
error / failed / denied などのログを抽出
🤖 AI異常検知
IsolationForestを用いた異常IP検出
通常と異なるアクセス頻度を自動検知
🧪 サンプルデータ

sample_log.csv を使用することで、すぐに動作確認が可能です。

以下のような攻撃パターンを含んでいます：

ブルートフォース攻撃
異常なアクセス頻度
SQLインジェクション風リクエスト
パストラバーサル攻撃
