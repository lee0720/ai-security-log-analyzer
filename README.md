# AI Security Log Analyzer

## 概要
サーバーログを解析し、不正アクセスや異常な通信を検知するツールです。

## 背景
ログ分析は手動では困難であり、異常の見逃しが発生するため、自動化を目的に開発しました。

## 機能
- IPアドレス別アクセス分析
- エラーログ抽出
- IsolationForestによる異常検知

## 技術
- Python
- Streamlit
- Pandas
- Scikit-learn

## 工夫点
- ログ形式に依存しない柔軟な解析
- AIによる外れ値検知を導入

「ログの中から“異常なアクセスパターン”を見つけるために、
単純な閾値ではなくIsolationForestを使いました」

👉
